\section{Генетический алгоритм} \label{ch2:title-abbr} %название по-русски
Генетические алгоритмы (ГА) - класс методов оптимизации, вдохновленных принципами естественного отбора и генетики.\\
Основная идея ГА заключается в том, чтобы эмулировать естественный процесс эволюции, в котором лучшие особи имеют больше шансов передать свои характеристики следующему поколению.
Путем применения операторов скрещивания, мутации и отбора, ГА итеративно улучшают текущее поколение решений, стремясь к нахождению оптимального или приближенного к нему решения.

\section{Метод роя частиц}
Метод роя частиц представляет собой метаэвристический алгоритм оптимизации, который основан на моделировании поведения роя в природе.\\
Основная идея PSO заключается в моделировании каждого потенциального решения (частицы) как точки в многомерном пространстве. Эти частицы перемещаются по пространству поиска с целью нахождения наилучшего решения, основываясь на лучшем решении, которое было найдено ими сами и другими частицами в прошлом.

\subsection{Подбор параметров $\omega, \phi_p, \phi_g$}
Метод роя частиц позволяет использовать различные стратегии подбора параметров $\omega, \phi_p, \phi_g$ для улучшения сходимости.
Используем следующие стратегии подбора для каждого из параметров соответственно:
\begin{itemize}
	\item $\omega$ - методом экспоненциального уменьшения:
	\begin{equation}\label{eq:exp_decay}
		\omega = (\omega_{0} - \omega_{term} - d_1)\exp{\bigg(\frac{1}{1 + \frac{d_2\cdot I}{I_{\max}}}\bigg)}
	\end{equation}
	Здесь $d_1 = 2, \ d_2 = 7, \omega_{term} = 0.4$.
	\item $\varphi_p$ - методом нелинейной модуляции:
	\begin{equation}\label{eq:nonlin_mod}
		\varphi_p = \varphi_{p_{term}} + (\varphi_{p_{0}} - \varphi_{p_{term}}) \cdot \bigg(\frac{
		I_{\max} - I}{I_{\max}}\bigg)^n
	\end{equation}
	Здесь $n = 1.2, \varphi_{p_{term}} = 0.8 \cdot \varphi_{p_{0}}$.
	\item $\varphi_g$ - методом линейной вариации:
	\begin{equation}\label{eq:lin_variation}
		\varphi_g = \varphi_{g_{term}} +  (\varphi_{g_{0}} - \varphi_{g_{term}}) \cdot \frac{I_{\max} - I}{I_{\max}}
	\end{equation}
	Здесь $\varphi_{g_{term}} = \varphi_{g_{0}}$.
\end{itemize}
под $I$ понимается номер текущей итерации, $I_{\max}$ - максимальное кол-во итераций.

\section{CMA-ES}
CMA-ES (Covariance Matrix Adaptation Evolution Strategy) - это метод оптимизации, который использует эволюционные стратегии для нахождения оптимальных решений в непрерывных пространствах параметров. Основная идея заключается в том, чтобы адаптировать ковариационную матрицу для оценки распределения параметров и эффективно обновлять вектор средних значений. При этом основополагающими концепциями являются нормальное многомерное распределение для моделирования параметров и ковариационная матрица для учета зависимостей между ними.
\subsection{Задание начальных значений}
Поясним некоторые обозначения:
\begin{itemize}
	\item $k \sim \mathcal{N}(0, C) \ \text{для} \ k = 1, \ldots, \lambda$ - векторы, распределенные по нормальному закону с матожиданием $0$ и матрицей ковариаций $C$.
	\item $B, D$ - результаты собственного разложения $C = BD^2B^T = BDDB^T$.
	\item  $x_k \in \mathbb{R}^n, k = 1, \ldots, \lambda$ - выборка размером $\lambda$ частиц.
	\item $\langle y \rangle_w = \sum_{i=1}^{\mu}\omega_iy_{i:\lambda}$ - размер взвешенного среднего без учёта $\sigma$.
	\item $y_{i:\lambda} = \frac{(x_{i:\lambda} - m)}{\sigma}$. $x_{i:\lambda} - m$ - определено ниже.
	\item $x_{i:\lambda} \in \mathbb{R}^n$ - $i$-ая лучшая позиция из $x_k = m + \sigma y_k, \quad \sim \mathcal{N}(m, \sigma^2C)$.\\
	Индекс $i$ указывает на $i$-ю ранжированную позицию из $f(x_{1:\lambda}) \leq f(x_{2:\lambda}) \leq \ldots \leq f(x_{\lambda:\lambda})$.
	\item $\mu = |\{\omega_i| \omega_i > 0\}|$ - количество строго положительных весов рекомбинации.
	\item $\mu_{eff} = \bigg( \sum_{i=1}^{\mu}\omega_i^{2}\bigg)^{-1}$ - среднее гармоническое (обратное среднее квадратов) весов $\omega_i$.
	\item $\displaystyle E||\mathcal{N}(0,I)|| = \sqrt{2}\frac{\Gamma(\frac{n+1}{2})}{\Gamma(\frac{n}{2})}$
	\item $h_\sigma$ - функция Хевисайда.
	\item $\delta(h_{\sigma}) -$ параметр малозначимости.
\end{itemize}

\section{Модель разумного водителя Трайбера}
Движение транспортного потока в реализации модели базируется на модели М. Трайбера "разумного водителя" (Intelligent Driver Model, IDM):
\begin{equation}
	s_{n}^{''}(t) = F(s_{n+1}(t) - s_{n}(t), s_{n+1}^{'}(t), s_{n}^{'}(t))
\end{equation}
\begin{equation}
	s_n^{''}(t) = a_n \bigg(1 - \bigg(\frac{s_n{'}(t)}{V_n^0}\bigg)^\delta - \bigg( 
	\frac{d_{n}^{*}(s_{n}^{'}(t), s_{n+1}^{'}(t)-s_{n}^{'}(t))}{s_{n+1}(t)-s_{n}(t)}\bigg)^2\bigg)
\end{equation}
Здесь:
\begin{itemize}
	\item Параметр $\delta$ отвечает за поведение при разгоне.
	\item $V_n^0$ - желаемая скорость $n$-го ТС.
	\item $d_{n}^{*}$ - желаемая поддерживаемая дистанция $n$-го ТС.
\end{itemize}
